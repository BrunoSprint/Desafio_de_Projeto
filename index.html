<h1> até <h6>:

<h1>: Cabeçalho de nível 1, é usado para títulos principais ou cabeçalhos de seção de maior importância em uma página.

<h2>: Cabeçalho de nível 2, é usado para títulos secundários ou cabeçalhos de seção que são um pouco menos importantes que os <h1>.

<h3>: Cabeçalho de nível 3, usado para títulos de subseções ou cabeçalhos de menor importância em relação aos <h2>.

<h4>: Cabeçalho de nível 4, utilizado para títulos de subseções ou informações menos relevantes que os níveis anteriores.

<h5>: Cabeçalho de nível 5, geralmente usado para informações detalhadas ou subtítulos de menor importância.

<h6>: Cabeçalho de nível 6, é o menos importante na hierarquia de cabeçalhos. É usado para informações ou subtítulos menos relevantes em relação aos níveis superiores.

<p>

A marcação <p> é uma tag HTML que representa um parágrafo em uma página web. Ela é amplamente utilizada por desenvolvedores para estruturar o conteúdo textual de uma página HTML.

Voltar

<mark>

A marcação <mark> é uma tag HTML usada para destacar partes do texto em uma página web. Essa tag é especialmente útil quando você deseja chamar a atenção do usuário para determinadas palavras ou trechos no conteúdo, sem alterar a estrutura semântica do texto.

Voltar

<small>

A marcação <small> é uma tag HTML usada para indicar que o texto contido deve ser renderizado em um tamanho de fonte menor do que o texto circundante. Essa tag é frequentemente utilizada para inserir textos de nota de rodapé, legendas, copyright, ou qualquer conteúdo secundário que seja relevante, mas que não precisa ser visualmente proeminente.

Voltar

<li>

A tag <li> é uma tag HTML usada para criar elementos de lista. Essa tag é comumente utilizada dentro de elementos <ul> (lista não ordenada) e <ol> (lista ordenada) para representar itens individuais na lista.

Voltar

<u>

A tag <u> é uma tag HTML utilizada para sublinhar o texto que a envolve. Essa tag foi historicamente usada para criar sublinhado em navegadores, indicando que o texto sublinhado é um hiperlink. No entanto, com a evolução das práticas de design e acessibilidade, o uso da tag <u> para criar sublinhado em textos que não são links tornou-se menos comum.

Voltar

<strong>

A tag <strong> é uma tag HTML usada para enfatizar ou destacar o texto, indicando que o texto contido deve ser exibido em negrito. A ênfase do <strong> é mais forte do que a ênfase proporcionada pela tag <em>, que é usada para enfatizar o texto de forma mais genérica ou enfática.

Voltar

<ol>

A tag <ol> é uma tag HTML usada para criar listas ordenadas, ou seja, listas nas quais os itens são numerados ou ordenados de alguma forma. Cada item da lista é representado pela tag <li> (de lista) aninhada dentro da tag <ol>.

Voltar

<ul>

A tag <ul> é uma tag HTML utilizada para criar listas não ordenadas, nas quais os itens não seguem uma ordem específica e não são numerados. Cada item da lista é representado pela tag <li> (item de lista) aninhada dentro da tag <ul>.

Voltar

<a>

A tag <a> é uma marcação HTML utilizada para criar links em uma página da web. Aqui estão alguns termos técnicos associados a esta tag: Elemento de Âncora (Element): A tag <a> é frequentemente chamada de "elemento de âncora" porque é usada para criar links de âncora em uma página. Atributo HREF (Hypertext Reference): A tag <a> geralmente inclui um atributo chamado "href", que especifica o destino do link. O valor do atributo "href" é o URL para o qual o link direcionará o usuário. Alvo do Link (Target): O atributo "target" pode ser utilizado para especificar como o link será aberto. Por exemplo, abrir o link em uma nova janela ou guia.

Voltar

<hr>

A marcação <hr> é utilizada em HTML para criar uma linha horizontal, muitas vezes usada para separar conteúdos em uma página da web. O elemento <hr> não possui uma tag de fechamento, pois é um elemento vazio, significando que não contém conteúdo entre uma tag de abertura e fechamento.

Voltar

<sup>

A tag <sup> é uma marcação em HTMLsup que é utilizada para representar texto ou elementos que devem ser exibidos como sobrescrito (superior) em relação à linha de texto normal. Geralmente, é utilizado para representar expoentes em equações matemáticas, números ordinais ou outras informações que devem aparecer acima do texto base.

Voltar

<sub>

A tag <sub> é uma marcação em HTMLsub que é usada para representar texto ou elementos que devem ser exibidos como subscrito (inferior) em relação à linha de texto normal. Essa tag é frequentemente utilizada para exibir índices, fórmulas químicas, números ordinais, ou qualquer outra informação que deva aparecer abaixo do texto base.

Voltar

<blockquote>

A tag <blockquote>

Em HTML é utilizada para representar um bloco de texto que é uma citação longa ou um bloco de texto retirado de outra fonte. Normalmente, os navegadores aplicam uma formatação específica para destacar visualmente o bloco de citação, como recuo nas margens.

Voltar

<font>

A tag <font> era uma tag obsoleta em HTML, mas ela costumava ser usada para especificar o tamanho, cor e face da fonte para o texto dentro do elemento ao qual era aplicada. Entretanto, seu uso foi desencorajado e eventualmente removido nas versões mais recentes do HTML, e a formatação de estilos de texto é mais apropriadamente realizada através de CSS (Cascading Style Sheets).

Voltar

<del>

A tag <del> em HTML é utilizada para representar texto que foi removido ou deletado de um documento. Quando o navegador renderiza o conteúdo da página, o texto dentro da tag <del> geralmente é exibido com uma linha de marcação (por exemplo, uma linha através do texto) para indicar que o conteúdo foi excluído. Além disso, a tag <del> é muitas vezes usada em conjunto com a tag , que representa texto inserido ou adicionado a um documento. Juntas, essas tags podem indicar as alterações entre diferentes versões de um documento.

Voltar

<abbr>

A tag <abbr> em HTML é utilizada para definir uma abreviação ou um acrônimo e fornece uma forma de incluir uma descrição completa ou expansão da abreviação. Isso é útil para acessibilidade, uma vez que permite que os leitores de tela e outros dispositivos interpretem e apresentem a forma completa da abreviação. O uso da tag <abbr> ajuda a melhorar a compreensão do conteúdo para usuários que podem não estar familiarizados com certas abreviações ou acrônimos, contribuindo para uma experiência mais acessível e informativa.
