Surgimento da Internet

Voltar

O surgimento da Internet é um marco fundamental na história da tecnologia da informação e comunicação, como a Internet se desenvolveu ao longo do tempo:

Década de 1960: O Início da ARPANET:
- A origem da Internet remonta à década de 1960, quando a ARPA Advanced Research Projects Agency dos Estados Unidos desenvolveu a ARPANET. Essa rede experimental foi criada para facilitar a comunicação e o compartilhamento de recursos entre pesquisadores e cientistas em diferentes instituições acadêmicas.

Protocolo TCP/IP:
- Na década de 1970, o desenvolvimento do protocolo TCP/IP Transmission Control Protocol/Internet Protocol se tornou crucial. Esse conjunto de protocolos padronizou a comunicação entre computadores em redes e é a base da Internet moderna.

Década de 1980, Expansão e Adoção Global:
- A ARPANET continuou a crescer e outras redes foram desenvolvidas. A proliferação de redes acadêmicas, governamentais e corporativas levou à necessidade de um sistema comum de comunicação, impulsionando a adoção do TCP/IP como padrão.

World Wide Web WWW:
- O desenvolvimento da World Wide Web, proposto por Tim Berners-Lee em 1989, foi um marco essencial. A WWW trouxe uma interface gráfica para a Internet, facilitando a navegação e o compartilhamento de informações.

Década de 1990: Popularização e Comercialização:
- Durante esta década, a Internet se tornou acessível ao público em geral. O crescimento explosivo da World Wide Web e o surgimento de provedores de serviços de Internet ISPs tornaram a Internet uma ferramenta de comunicação e pesquisa acessível a milhões de pessoas em todo o mundo.

Década de 2000 até o Presente: Evolução Contínua:
- A Internet continuou a evoluir com o tempo. Houve avanços significativos em velocidade de conexão, segurança, protocolos de comunicação e desenvolvimento de conteúdo. Surgiram redes sociais, serviços em nuvem, comércio eletrônico e uma miríade de aplicativos e serviços online.

Hoje, a Internet é uma parte essencial da vida moderna, afetando quase todos os aspectos da sociedade, negócios e comunicação. A capacidade de conectar pessoas, dispositivos e dados em escala global tornou-se uma característica distintiva da era digital.

Como Funciona o HTML

Voltar

O HTML, ou Hypertext Markup Language Linguagem de Marcação de Hipertexto, em português, é uma linguagem padrão usada para criar e estruturar conteúdo em páginas da web. Ele fornece a estrutura básica para os documentos web, indicando como o conteúdo deve ser organizado e apresentado nos navegadores.

O HTML utiliza uma abordagem de "marcação" para envolver o conteúdo com tags específicas que indicam o significado semântico dos elementos. Cada tag possui uma função única, como definir parágrafos, cabeçalhos, links, imagens, formulários, entre outros.

Um documento HTML começa com uma declaração que informa ao navegador sobre a versão do HTML que está sendo usada. Em seguida, o conteúdo é envelopado pelas tags `html`, que consistem em duas seções principais: `head` e `body`. A seção `head` contém informações de metadados, como o título da página, links para folhas de estilo CSS, e outros elementos que não são diretamente visíveis na página. A seção `body` contém o conteúdo visível na página, como texto, imagens, links e outros elementos.

Além disso, o HTML é uma linguagem extensível, o que significa que é possível incorporar outras tecnologias, como CSS para estilização e JavaScript para adicionar interatividade. Isso permite criar páginas web visualmente atraentes e dinâmicas.

A evolução contínua do HTML tem sido liderada pelo World Wide Web Consortium W3C e pela Web Hypertext Application Technology Working Group WHATWG. A última versão principal é o HTML5, que introduziu novos elementos semânticos, APIs Application Programming Interfaces e recursos para melhorar a experiência de desenvolvimento e a capacidade da web.

Em resumo, o HTML é a espinha dorsal da web, fornecendo a estrutura fundamental para a criação de documentos interativos e acessíveis, permitindo a comunicação eficaz de informações na internet.

O que são Tags

Voltar

Em contexto tecnológico, o termo "tags" geralmente se refere a elementos específicos em linguagens de marcação, como HTML Hypertext Markup Language ou XML Extensible Markup Language.

Em HTML, as tags são elementos fundamentais usados para estruturar e definir o conteúdo de uma página web. Elas são cercadas por colchetes angulares e geralmente ocorrem em pares de abertura e fechamento, indicando onde um elemento começa e onde termina.

As tags também podem conter atributos, que fornecem informações adicionais sobre o elemento. A sintaxe básica inclui o nome do atributo seguido pelo sinal de igual `=` e o valor do atributo, envolto por aspas simples ou duplas.

É possível aninhar tags, ou seja, colocar uma tag dentro de outra. Isso cria uma hierarquia que define a estrutura do conteúdo. Exemplo com um parágrafo dentro de uma lista

Neste caso, `ul` é a tag de abertura da lista não ordenada, li é a tag de abertura para cada item da lista, `/li` e `/ul` são as tags de fechamento correspondentes.

Com o HTML5, foram introduzidos elementos semânticos que dão significado mais claro ao conteúdo. Exemplos incluem `header`, `nav`, `article`, `section`, `footer`, entre outros.

Em resumo, as tags em HTML são elementos cruciais que ajudam a estruturar e formatar o conteúdo em uma página web, permitindo aos desenvolvedores criar documentos web semânticos e interativos.
